appforpid

api.app.appforpid(pid) -> app or nil

Returns the running app for the given pid, if it exists.

appsforbundleid

api.app.appsforbundleid(bundleid) -> app[]

Returns any running apps that have the given bundleid.

launchorfocus

api.app.launchorfocus(name)

Launches the app with the given name, or activates it if it's already running.

runningapps

api.app.runningapps() -> app[]

Returns all running apps.

activate

api.app:activate() -> bool

Tries to activate the app (make it focused) and returns its success.

allwindows

api.app:allwindows() -> window[]

Returns all open windows owned by the given app.

bundleid

api.app:bundleid() -> string

Returns the bundle identifier of the app.

hide

api.app:hide()

Hides the app (and all its windows).

ishidden

api.app:ishidden() -> bool

Returns whether the app is currently hidden.

kill

api.app:kill()

Tries to terminate the app.

kill9

api.app:kill9()

Assuredly terminates the app.

show

api.app:show()

Unhides the app (and all its windows) if it's hidden. Yeah, this method name should probably be renamed.

title

api.app:title() -> string

Returns the localized name of the app (in UTF8).

visiblewindows

api.app:visiblewindows() -> win[]

Returns only the app's windows that are visible.
