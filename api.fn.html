api fn

Super-helpful functional programming utilities.

concat

api.fn.concat(t1, t2)
Adds all elements of t2 to the end of t1.

contains

api.fn.contains(t, el) -> bool
Returns whether a table contains a given element.

filter

api.fn.filter(t, fn) -> t
Returns a table of the elements in t in which t(el) is truthy.

indexof

api.fn.indexof(t, el) -> int or nil
Returns the index of a given element in a table, or nil if not found.

map

api.fn.map(t, fn) -> t
Returns a table of the results of t(el) on every el in t.

mapcat

api.fn.mapcat(t, fn) -> t2
Runs fn(el) for every el in t, and assuming the results are tables, combines them into a new table.

reduce

api.fn.reduce(t, fn) -> t2
Runs fn(el1, el2) for every el in t, then fn(result, el3), etc, until there's only one left.
