hydra

General stuff.

alert

hydra.alert(str, seconds = 2)

Shows a message in large words briefly in the middle of the screen; does tostring() on its argument for convenience..

call

hydra.call(fn, ...) -> ...

Just like pcall, except that failures are handled using hydra.errorhandler

check_accessibility

hydra.check_accessibility(shouldprompt) -> isenabled

Returns whether accessibility is enabled. If passed true, prompts the user to enable it.

douserfile

hydra.douserfile(name)

Convenience wrapper around dofile() and hydra.userfile(name)

errorhandler

hydra.errorhandler = function(err)

Error handler for hydra.call; intended for you to set, not for third party libs

exec

hydra.exec(command) -> string

Runs a shell function and returns stdout as a string (without trailing newline).

fileexists

hydra.fileexists(path) -> exists, isdir

Checks if a file exists, and whether it's a directory.

focushydra

hydra.focushydra()

Makes Hydra the currently focused app; useful in combination with textgrids.

indock

hydra.indock() -> bool

Returns whether Hydra has a Dock icon, and thus can be switched to via Cmd-Tab.

putindock

hydra.putindock(bool)

Sets whether Hydra has a Dock icon, and thus can be switched to via Cmd-Tab.

reload

hydra.reload()

Reloads your init-file. Makes sure to clear any state that makes sense to clear (hotkeys, pathwatchers, etc).

resourcesdir

hydra.resourcesdir -> string

The path of the built-in lua source files, with no trailing slash.

showabout

hydra.showabout()

Displays the standard OS X about panel; implicitly focuses Hydra.

userfile

hydra.userfile(name)

Returns the full path to the file ~/.hydra/{name}.lua

uuid

hydra.uuid() -> string

Returns a UUID as a string
