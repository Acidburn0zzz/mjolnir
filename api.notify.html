api notify

Apple's built-in notifications system.

  • api.notify.register
  • api.notify.show
  • api.notify.unregister
  • api.notify.unregisterall

register

api.notify.register(tag, fn()) -> id
Registers a function to be called when an Apple notification with the given tag is clicked.

show

api.notify.show(title, subtitle, text, tag)
Show an Apple notification. Tag is a unique string that identifies this notification, and will be passed to api.notify.clicked() if the notification is clicked. None of the strings are optional, though they may each be blank.

unregister

api.notify.unregister(id)
Unregisters a function to no longer be called when an Apple notification with the given tag is clicked.

unregisterall

api.notify.unregisterall()
Unregisters all functions registered for notification-clicks; called automatically when user config reloads.
