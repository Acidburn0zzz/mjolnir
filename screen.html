screen

Manipulate monitors (aka screens).

allscreens

screen.allscreens() -> screen[]

Returns all the screens there are.

frame

screen.frame(screen) -> rect

Returns a screen's frame in its own coordinate space.

mainscreen

screen.mainscreen() -> screen

Returns the 'main' screen, i.e. the one containing the currently focused window.

settint

screen.settint(redarray, greenarray, bluearray)

Set the tint on a screen; experimental.

vislbleframe

screen.visibleframe(screen) -> rect

Returns a screen's frame in its own coordinate space, without the dock or menu.

frame_including_dock_and_menu

screen:frame_including_dock_and_menu() -> rect

Returns the screen's rect in absolute coordinates, including the dock and menu.

frame_without_dock_or_menu

screen:frame_without_dock_or_menu() -> rect

Returns the screen's rect in absolute coordinates, without the dock or menu.

next

screen:next() -> screen

Returns the screen 'after' this one; I have no idea how they're ordered though.

previous

screen:previous() -> screen

Returns the screen 'before' this one; I have no idea how they're ordered though.
