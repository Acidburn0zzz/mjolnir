api.log

Functionality to assist with debugging and experimentation.

addhandler

api.log.addhandler(fn(str)) -> index

Registers a function to handle new log lines.

lines

api.log.lines = {}

List of lines logged so far; caps at api.log.maxlines. You may clear it by setting it to {} yourself.

maxlines

api.log.maxlines = 500

Maximum number of lines to be logged.

removehandler

api.log.removehandler(index)

Unregisters a function that handles new log lines.

show

api.log.show() -> textgrid

Opens a textgrid that can browse all logs.
