hotkey

Manage global hotkeys.

bind

hotkey.bind(...) -> hotkey

Shortcut for: return hotkey.new(...):enable()

disableall

hotkey.disableall()

Disables all hotkeys; automatically called when user config reloads.

new

hotkey.new(mods, key, fn) -> hotkey

Creates a new hotkey that can be enabled. Mods is a table containing any of the elements {cmd, ctrl, alt, shift}. Key may be any 1-character string, or 'F1', 'Enter', etc. Both are case-insensitive. The hotkey has the public fields: key, mods, fn.

disable

hotkey:disable() -> self

Disables the given hotkey; does not remove it from hotkey.keys.

enable

hotkey:enable() -> hotkey

Registers the hotkey's fn as the callback when the user presses key while holding mods.
